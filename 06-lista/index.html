Listas com Bootstrap 5

O Bootstrap 5 oferece uma variedade de classes para estilizar listas, tornando-as mais atraentes e fáceis de ler. Você pode usar as classes .list-group para criar listas de itens, e as classes .list-group-item para estilizar cada item da lista. Além disso, o Bootstrap suporta listas ordenadas, não ordenadas e de descrição.
