Botões com Bootstrap 5

O Bootstrap 5 oferece uma variedade de estilos e tipos de botões para atender a diferentes necessidades de design e funcionalidade.

1. Tipos de Botões

Os botões no Bootstrap 5 podem ser criados usando a classe .btn junto com uma classe de estilo, como:

  • .btn: Estilo base para botões.
  • .btn-primary: Botão principal, geralmente usado para ações principais.
  • .btn-secondary: Botão secundário, usado para ações menos importantes.
  • .btn-success: Botão de sucesso, usado para indicar ações bem-sucedidas.
  • .btn-danger: Botão de perigo, usado para indicar ações perigosas ou destrutivas.
  • .btn-warning: Botão de aviso, usado para indicar ações que requerem atenção.
  • .btn-info: Botão de informação, usado para indicar ações informativas.
  • .btn-light: Botão claro, usado para ações neutras.
  • .btn-dark: Botão escuro, usado para ações mais fortes ou contrastantes.
