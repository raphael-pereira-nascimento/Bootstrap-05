Dropdown

Os Dropdowns são menus sobrepostos e alternáveis que exibem listas de links ou ações.

1. Estrutura Básica

Um dropdown padrão precisa de um "pai" com a classe .dropdown e dois elementos internos:

  • O Gatilho (Botão/Link): Deve ter a classe .dropdown-toggle e o atributo data-bs-toggle="dropdown".
  • O Menu (<ul>): Deve ter a classe .dropdown-menu.

2. Elementos Internos do Menu

Para deixar o menu organizado, o Bootstrap oferece classes utilitárias:

  • .dropdown-item: Estiliza cada link da lista.
  • .dropdown-divider: Cria uma linha horizontal para separar grupos de ações (como "Configurações" de "Sair").
  • .dropdown-header: Um rótulo de texto não clicável para categorizar os itens.
  • .active / .disabled: Para indicar o estado do item.

3. Posicionamento (Direção)

Você pode controlar para onde o menu abre usando classes no container pai:

  • .dropup: Abre para cima.
  • .dropend: Abre para a direita.
  • .dropstart: Abre para a esquerda.

Dropup

O Dropup é exatamente o que o nome sugere: um menu que, em vez de abrir para baixo, abre para cima. Ele é muito útil em rodapés (footers) ou em barras de ferramentas posicionadas na parte inferior da tela, evitando que o menu fique escondido para fora da janela do navegador. Para transformar um Dropdown comum em um Dropup, você só precisa trocar a classe .dropdown no elemento pai pela classe .dropup.
