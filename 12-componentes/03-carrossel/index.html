Carrossel com Bootstrap 5

O Bootstrap 5 oferece um componente de carrossel para criar sliders de imagens ou conteúdo de forma fácil e responsiva.

1. Principais Recursos

  • Controles de navegação simples
  • Indicadores visuais
  • Transições suaves
  • Responsividade

2. Classes

As classes principais do carrossel são:

  • carousel
  • carousel-inner
  • carousel-item
  • carousel-control-prev
  • carousel-control-next
  • carousel-indicators
  • carousel-caption
  • carousel-fade

Intervalo Customizado: Você pode definir o tempo de cada slide individualmente usando data-bs-interval="2000" (em milissegundos).


Com a classe .slide (Padrão)

.slide

Quando você mantém a classe <div class="carousel slide">, o Bootstrap aplica uma transição suave de deslizamento (animação CSS) entre um item e outro. É o efeito clássico que a maioria dos usuários espera de um carrossel.

Sem a classe .slide

.slide

Se você remover a classe e deixar apenas <div class="carousel">, a troca de imagens será instantânea. O slide simplesmente "pisca" e muda para o próximo, sem nenhum movimento lateral.

3. Estrutura Básica (O Mínimo para Funcionar)

Um carrossel precisa de um ID exclusivo e de três camadas de HTML:

O Container (.carousel): Com o atributo data-bs-ride="carousel".

A Moldura (.carousel-inner): Onde os itens ficam.

Os Itens (.carousel-item): Cada slide individual.

Atenção: Pelo menos um item deve ter a classe (.active)
