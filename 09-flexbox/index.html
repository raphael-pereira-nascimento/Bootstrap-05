Flexbox com Bootstrap 5

O Flexbox é um modelo de layout poderoso que permite criar layouts flexíveis e responsivos. O Bootstrap 5 inclui uma série de classes utilitárias para trabalhar com Flexbox, facilitando a criação de layouts complexos sem a necessidade de escrever CSS personalizado.

1. Container Flexível

Para criar um container flexível, basta adicionar a classe .d-flex ao elemento pai. Isso transforma o elemento em um contêiner flexível, permitindo que seus filhos sejam organizados usando as propriedades do Flexbox.

3. Alinhamento dos Itens

O alinhamento dos itens flexíveis pode ser controlado usando as classes de alinhamento do Flexbox. As opções incluem: .justify-content-start, .justify-content-end, .justify-content-center, .justify-content-between e .justify-content-around.
