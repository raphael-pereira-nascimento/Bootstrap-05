Os três tipos de Containers

O Bootstrap 5 oferece três tipos de containers para organizar o layout da sua página: .container, .container-fluid e .container-{breakpoint}.

1. Container Padrão (.container)

O container padrão é responsivo e tem uma largura fixa que se ajusta de acordo com o tamanho da tela. Ele é centralizado horizontalmente e é ideal para a maioria dos layouts.

Este é um exemplo de container padrão. Ele tem uma largura fixa e é centralizado na página.

2. Container Fluido (.container-fluid)

O container fluido ocupa toda a largura da tela, independentemente do tamanho do dispositivo. Ele é útil para layouts que precisam se estender por toda a largura da janela.

Este é um exemplo de container fluido. Ele ocupa toda a largura da tela, sem margens laterais.

3. Container Responsivo por Breakpoint (.container-{breakpoint})

O container responsivo por breakpoint é uma variação do container padrão que se ajusta a largura com base no ponto de interrupção especificado. Os breakpoints disponíveis são: sm, md, lg, xl e xxl.
