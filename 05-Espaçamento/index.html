Exemplos de Padding (Espaçamento Interno)

O Bootstrap oferece uma variedade de classes de espaçamento para controlar o padding (espaçamento interno) dos elementos. Essas classes seguem a convenção .p{direção}-{tamanho},onde direção pode ser:

  • t (Topo), b (base), s (inicio), e (fim), x (horizontal) e y (vertical).
  • {tamanho} varia de 0 a 5,onde 0 é sem padding e 5 é o maior padding.
