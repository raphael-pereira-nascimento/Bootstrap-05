Exemplos de Padding (Espaço Interno)

O Bootstrap oferece uma variedade de classes de espaçamento para controlar o padding (espaço interno) dos elementos. Essas classes seguem a convenção .p{direção}-{tamanho}, onde direção pode ser:

  • t (topo), b (base), s (início), e (fim), x (horizontal) e y (vertical).
  • {tamanho} varia de 0 a 5, onde 0 é sem padding e 5 é o maior padding.

.p-0

Sem padding, o conteúdo fica colado nas bordas do elemento.

.p-1

Padding pequeno, ideal para elementos compactos ou para criar um leve espaçamento.

.p-2

Padding médio, ótimo para a maioria dos casos, proporcionando um bom equilíbrio entre conteúdo e espaço.

.p-3

Padding grande, ideal para destacar seções importantes ou para criar um layout mais arejado.

.p-4

Padding extra grande, perfeito para criar áreas de destaque ou para separar visualmente seções importantes.

.p-5

Padding máximo, ideal para criar seções muito destacadas ou para dar um ar mais "clean" e espaçado ao layout.

Exemplos de Margin (Espaço Externo)

As classes de margin seguem a mesma convenção das classes de padding, mas controlam o espaço externo dos elementos. Elas são usadas para criar espaçamento entre elementos, evitando que fiquem colados uns aos outros.

.m-0

Sem margin, os elementos ficam colados uns aos outros, o que pode criar um layout confuso.

.m-1

Margin pequeno, ideal para criar um leve espaçamento entre elementos próximos.

.m-2

Margin médio, ótimo para a maioria dos casos, proporcionando um bom equilíbrio entre elementos.

.m-3

Margin grande, ideal para destacar seções importantes ou para criar um layout mais arejado.

.m-4

Margin extra grande, perfeito para criar áreas de destaque ou para separar visualmente seções importantes.

.m-5

Margin máximo, ideal para criar seções muito destacadas ou para dar um ar mais "clean" e espaçado ao layout.

Exemplos de Padding e Margin Combinados

Você também pode combinar as classes de padding e margin para criar layouts mais complexos e personalizados. Por exemplo, usando .p-3 para o padding interno e .m-2 para o margin externo, você pode criar um elemento que tenha um bom espaçamento interno e externo ao mesmo tempo.

.p-3 + .m-2

Este elemento tem um padding grande e um margin médio, criando um layout mais arejado e organizado.

.p-4 + .m-3

Este elemento tem um padding extra grande e um margin grande, ideal para destacar seções importantes ou criar áreas de destaque.

.p-5 + .m-4

Este elemento tem o padding máximo e um margin extra grande, perfeito para criar seções muito destacadas ou para dar um ar mais "clean" e espaçado ao layout.

.p-2 + .m-1

Este elemento tem um padding médio e um margin pequeno, ideal para criar um layout equilibrado e organizado.

.p-1 + .m-0

Este elemento tem um padding pequeno e sem margin, criando um layout mais compacto e próximo.

.p-0 + .m-5

Este elemento tem sem padding e o margin máximo, ideal para criar um layout muito espaçado e arejado, onde os elementos ficam bem separados uns dos outros.

Exemplos de Padding e Margin Específicos

Além das classes genéricas de padding e margin, o Bootstrap também oferece classes específicas para cada direção, como .pt-3 para padding-top ou .mb-2 para margin-bottom. Isso permite um controle ainda mais preciso sobre o espaçamento dos elementos.

.pt-3

Padding-top grande, ideal para criar um espaçamento maior na parte superior do elemento.

.mb-2

Margin-bottom médio, ótimo para criar um espaçamento equilibrado na parte inferior do elemento.

.ps-4

Padding-start extra grande, perfeito para criar um espaçamento maior na parte esquerda do elemento (ou direita em layouts RTL).

.me-1

Margin-end pequeno, ideal para criar um leve espaçamento na parte direita do elemento (ou esquerda em layouts RTL).

.px-5

Padding-horizontal máximo, ideal para criar um espaçamento muito grande nas partes esquerda e direita do elemento.

.my-4

Margin-vertical extra grande, perfeito para criar um espaçamento maior nas partes superior e inferior do elemento.

.py-2

Padding-vertical médio, ótimo para criar um espaçamento equilibrado nas partes superior e inferior do elemento.

.pt-5 + .mb-3

Este elemento tem um padding-top máximo e um margin-bottom grande, ideal para criar um espaçamento muito grande na parte superior e um espaçamento grande na parte inferior do elemento.

.ps-1 + .me-4

Este elemento tem um padding-start pequeno e um margin-end extra grande, perfeito para criar um espaçamento leve na parte esquerda e um espaçamento muito grande na parte direita do elemento.

.px-3 + .my-1

Este elemento tem um padding-horizontal grande e um margin-vertical pequeno, ideal para criar um espaçamento grande nas partes esquerda e direita do elemento, e um espaçamento leve nas partes superior e inferior.

.py-4 + .mx-2

Este elemento tem um padding-vertical extra grande e um margin-horizontal médio, perfeito para criar um espaçamento muito grande nas partes superior e inferior do elemento, e um espaçamento equilibrado nas partes esquerda e direita.

.pt-1 + .mb-5

Este elemento tem um padding-top pequeno e um margin-bottom máximo, ideal para criar um espaçamento leve na parte superior e um espaçamento muito grande na parte inferior do elemento.

.ps-2 + .me-3

Este elemento tem um padding-start médio e um margin-end grande, ideal para criar um espaçamento equilibrado na parte esquerda e um espaçamento grande na parte direita do elemento.

.px-4 + .my-5

Este elemento tem um padding-horizontal extra grande e um margin-vertical máximo, perfeito para criar um espaçamento muito grande nas partes esquerda e direita do elemento, e um espaçamento máximo nas partes superior e inferior.
