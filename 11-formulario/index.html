Formulário com Bootstrap 5

1. Estrutura Básica de um Campo

Um campo de formulário no Bootstrap 5 é composto por um rótulo (<label>) e um input (<input> ou <textarea>).

2. Tipos de Campos Comuns

O Bootstrap 5 suporta diversos tipos de campos, como:

  • text: Campo de texto simples.
  • email: Campo para endereços de e-mail.
  • password: Campo para senhas, com caracteres ocultos.
  • textarea: Campo para texto multilinha.
  • select: Campo de seleção dropdown.

5. Tamanhos e Estados

O Bootstrap 5 oferece classes para ajustar o tamanho dos campos e indicar estados de validação.

readonly / disabled: Atributos HTML que o Bootstrap estiliza automaticamente para parecerem inativos.

6. Floating Labels (Rótulos Flutuantes)

O Bootstrap 5 introduziu os "Floating Labels", que são rótulos que flutuam acima do campo de entrada quando o usuário interage com ele. Isso cria uma experiência de formulário mais limpa e moderna.

Regra: O <input> deve vir antes do <label> e ambos precisam estar dentro de uma div .form-floating. O input precisa de um placeholder (mesmo que vazio).

7. Validação (Visual)

O Bootstrap 5 oferece classes para indicar visualmente o estado de validação dos campos. Essas classes são úteis para fornecer feedback ao usuário sobre a validade dos dados inseridos.
